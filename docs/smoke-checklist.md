# スモークチェックリスト（フェーズ出口ゲート）

各フェーズのデプロイ後に実施。375px / 768px / 1280px。フェーズごとに最低 1 回はスマホ実機
（実機の JS エラーは `/api/client-log` → Railway logs に届く。デプロイ後 24h はログ監視）。

## いいねタブ（grid）
- [ ] 初期ロードでタイルが表示される（スケルトン → 画像）
- [ ] 無限スクロールで次の 60 件が継ぎ足される
- [ ] 作者 / タグ / メディア種別 / 検索 / リストの各フィルタ + フィルタチップ表示
- [ ] リスト追加・解除（★ オーバーレイ反映）
- [ ] お気に入り作者トグル
- [ ] 投稿削除（タイルが消える・undo なし確認）
- [ ] lightbox: 開く / ◀▶ / ←→ キー / Esc / 作者リンク
- [ ] リールモード: 切替・スナップスクロール・動画自動再生

## フォロー中タブ（timeline）
- [ ] タブ切替で表示
- [ ] ⟳ 取得（クールダウン中は 429 通知が穏当に出る）
- [ ] 既読ライン（last-seen）+ 📍 ボタン
- [ ] いいね & 保存（ハート busy → done）
- [ ] いいね済みを隠すトグル
- [ ] プロキシ画像 / 動画が表示・再生される

## 本棚タブ
- [ ] カバー遅延読込
- [ ] お気に入り / タグフィルタ
- [ ] タグ編集・タイトル変更
- [ ] ファイルアップロード追加（複数ファイル順序）
- [ ] URL インポート + キュー状態表示・重複スキップ通知
- [ ] 削除

## リーダー
- [ ] 開く / RTL スワイプ / クリックナビ / ページ表示
- [ ] スライドアニメ / 閉じる（× と Esc）
- [ ] 背面スクロールしない

## ポップオーバー / モーダル
- [ ] ⚙ メニュー（ユーザー名保存・マイいいね同期・cookie 状態・バージョン表示）
- [ ] リストポップオーバー（画面端でクランプされる）
- [ ] cookie モーダル（小画面でスクロール可・保存/接続テストの結果がバナー表示）
- [ ] サイドバーハンバーガー開閉 + オーバーレイ

## グローバル
- [ ] 砂時計が同期/重複チェック中だけ出て消える
- [ ] 通知バナー: info 自動消滅 / error はタップで閉じる
- [ ] 10 分以内の再読込 → 自動同期が静かにスキップされる
- [ ] 手動 ⟳ 同期ボタンは常に動く
- [ ] `curl -sI` の X-App-Version が期待版
- [ ] デプロイ 24h 後 Railway logs に client-log エラーなし
