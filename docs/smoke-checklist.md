# スモークチェックリスト（フェーズ出口ゲート）

各フェーズのデプロイ後に実施。375px / 768px / 1280px。フェーズごとに最低 1 回はスマホ実機
（実機の JS エラーは `/api/client-log` → Railway logs に届く。デプロイ後 24h はログ監視）。

## PWA インストール (v0.6.3〜)
